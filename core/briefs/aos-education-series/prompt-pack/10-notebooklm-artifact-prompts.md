# NotebookLM Artifact Prompts

These prompts are designed for NotebookLM-generated artifacts after the notebook has been loaded with the AOS educational series sources.

## Report Prompt

```text
Create a report that synthesizes the AOS educational-series brief, editorial plan, and supporting repo documentation into a concise but concrete publishing strategy.

The report should include:

- what AOS Harness is in public-facing language
- who this series is for
- the strongest educational themes to emphasize
- the best ordering of topics for Season 1
- where hands-on examples matter most
- which topics need simplification before public publication
- a final recommendation for how to position the series

Keep the writing grounded in the loaded AOS sources.
```

## Study Guide Prompt

```text
Create a study guide for the AOS educational series so the author can internalize the core system concepts before writing.

Organize the guide into:

- foundational concepts
- structural concepts
- execution concepts
- continuity concepts
- customization concepts

For each section, define the key idea, why it matters, and what to teach readers first.
```

## Flashcards Prompt

```text
Create flashcards that help the author learn the most important distinctions in the AOS Harness architecture.

Focus on distinctions such as:

- agent vs profile
- profile vs workflow
- domain overlay vs domain enforcement
- deliberation vs execution
- memory vs expertise
- replay vs resumption
- orchestrator vs worker
```

## Quiz Prompt

```text
Create a quiz that tests whether someone actually understands AOS Harness well enough to write educational posts about it.

Focus on architecture, terminology, sequencing, and practical usage.
```

## Audio Overview Prompt

```text
Create an audio overview script for the AOS educational-series plan.

The script should sound like a guided walkthrough of:

- what AOS Harness is
- why the series exists
- how the season is structured
- what readers will learn over time

Keep the tone practical and technically grounded.
```
