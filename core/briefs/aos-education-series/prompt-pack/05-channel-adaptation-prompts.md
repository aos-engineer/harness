# Prompt Pack: Channel Adaptation

## LinkedIn Adaptation Prompt

```text
Adapt this AOS educational draft into a LinkedIn post.

Requirements:

- Strong hook in the first 1 to 2 lines
- Fast readability on mobile
- Professional but human tone
- Emphasize one core idea only
- End with a low-friction invitation to follow the series or read the next post
- Keep examples concrete
- Avoid sounding like a launch announcement or marketing campaign

Return:

- main LinkedIn version
- shorter version
- slightly more technical version
```

## Newsletter Adaptation Prompt

```text
Adapt this AOS educational draft into a newsletter entry.

Requirements:

- Preserve the same core lesson
- Add more context and explanatory depth
- Include one short "why this matters" section
- Include one short "try this" section
- End with a teaser for the next issue
- Keep the tone educational and grounded

Return:

- full newsletter version
- optional subtitle
- optional subject line options
```

## Blog Adaptation Prompt

```text
Expand this AOS educational draft into a blog post.

Requirements:

- Keep the same core argument
- Add one section explaining the repo concept in more detail
- Add one section connecting the concept to the larger AOS architecture
- Include one practical reader takeaway
- Include one soft bridge to the next article

Return:

- polished blog draft
- suggested headings
- suggested excerpt
```
