# Prompt: Post Drafting Master Prompt

Use this prompt to draft any single post from the season.

## Prompt Template

```text
Draft a public educational post about AOS Harness using the following inputs.

Inputs:

- Channel: [LinkedIn | newsletter | blog]
- Post title: [TITLE]
- Audience: [AUDIENCE SEGMENT]
- Core lesson: [CORE LESSON]
- Repo concept to reference: [AOS CONCEPT]
- Practical example: [EXAMPLE]
- Desired takeaway: [TAKEAWAY]
- Tiny action for reader: [ACTION]
- Teaser for next post: [NEXT TEASER]

Write the post with these goals:

1. Teach one clear idea only
2. Keep the writing grounded in how AOS Harness actually works
3. Make the reader feel that AI can be structured and learned, not just prompted
4. End with a clear takeaway and a reason to return

Style rules:

- Educational before promotional
- Confident, practical, and inviting
- No inflated AI claims
- No generic productivity advice
- Use plain language for new readers, but retain technical credibility
- If referencing AOS concepts, explain them in one sentence before going deeper

Structure:

- Opening hook
- Core explanation
- Why it matters in AOS Harness
- One practical example
- Takeaway
- Tiny action
- Bridge to next post

If the channel is LinkedIn:

- Keep it concise and readable in-feed
- Lead with the hook immediately
- Favor short paragraphs

If the channel is newsletter or blog:

- Expand the explanation and example
- Add one short section that gives more context

Return:

- A finished draft
- 3 alternate hooks
- 3 alternate closing lines
```
